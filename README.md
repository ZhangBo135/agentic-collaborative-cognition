# Agentic Collaborative Cognition Project Page

This folder is a static GitHub Pages-ready project page for:

**Agentic Collaborative Cognition for Zero-Shot 3D Understanding**

## Local Preview

From this folder, run:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Deployment

Upload the contents of this folder to a GitHub repository and enable GitHub Pages
for the branch that contains `index.html`.
